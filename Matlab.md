MATLAB 矩阵操作 超完整精讲
 
MATLAB 全称矩阵实验室，所有数据本质都是矩阵/数组，标量是1\times1矩阵，行向量是1\times n矩阵，列向量是n\times1矩阵。
核心规则：
 
1. 索引从 1 开始（区别于 Python/C++）；
2. 分行用 ; ，同行元素用空格/逗号分隔；
3. 分两大类：线性代数矩阵运算、逐元素点运算。
 
 
 
一、矩阵基础创建
 
1. 手动自定义矩阵
 
matlab
  

% 1. 行向量 1行n列
v1 = [1, 2, 3, 4];  
v1 = [1 2 3 4];     

% 2. 列向量 n行1列，分号换行
v2 = [1; 2; 3; 4];  

% 3. 二维矩阵 m行n列
A = [1 2 3; 
     4 5 6; 
     7 8 9];
 
 
2. 规律快速生成
 
（1）冒号表达式（最常用）
 
语法： 起始:步长:终止 
 
matlab
  

x = 1:5;        % 步长默认1，[1 2 3 4 5]
x = 1:2:9;      % 步长2，[1 3 5 7 9]
x = pi:0.5:2*pi;% 小数/浮点数序列
 
 
（2）等分序列
 
matlab
  

linspace(a,b,n)  % [a,b] 均匀生成 n 个点
x = linspace(0, 2*pi, 100); 

logspace(a,b,n)  % 对数等分
 
 
3. 预置特殊矩阵
 
matlab
  

zeros(m,n)      % m行n列 全0矩阵
ones(m,n)       % m行n列 全1矩阵
eye(n)          % n阶 单位方阵（对角线1，其余0）
rand(m,n)       % 0~1 均匀随机矩阵
randn(m,n)      % 标准正态分布随机矩阵
diag([1 2 3])   % 对角矩阵，向量转为对角方阵
magic(n)        % n阶幻方矩阵
 
 
 
 
二、矩阵索引 & 元素访问（核心重点）
 
1. 双下标索引（行,列）
 
最直观写法： 矩阵(行号, 列号) 
 
matlab
  

A = [1 2 3; 4 5 6; 7 8 9];
A(2,3)   % 第2行第3列，结果=6
A(1,1)   % 左上角第一个元素
 
 
2. 冒号批量索引
 
-  :  代表所有
-  a:b  代表第 a ～ b 个
 
matlab
  

A(2,:)     % 取第2行 全部列 → 行向量
A(:,3)     % 取第3列 全部行 → 列向量
A(1:2, :)  % 取1~2行，所有列
A(:, 2:3)  % 取所有行，2~3列
A(1:2, 2:3)% 截取子矩阵：1-2行、2-3列
 
 
3.  end  关键字
 
自动代表最后一行/最后一列，适配任意尺寸矩阵：
 
matlab
  

A(end, :)     % 最后一行
A(:, end)     % 最后一列
A(2:end, :)   % 第2行到最后一行
 
 
4. 单下标索引（列优先）
 
MATLAB 矩阵按列排序编号，适合简易取值：
 
matlab
  

A(5)   % 先数第1列、再第2列…第5个元素 = 5
 
 
5. 逻辑索引（高级筛选）
 
按条件提取矩阵元素：
 
matlab
  

A(A>5)   % 提取所有大于5的元素
A(mod(A,2)==0) % 提取所有偶数
 
 
 
 
三、矩阵修改、删除、赋值
 
1. 单个/批量元素赋值
 
matlab
  

A(2,2) = 99;       % 修改单个元素
A(1,:) = [10 20 30];% 整行覆盖赋值
 
 
2. 删除行/列
 
将选中部分赋值为空矩阵  [] ：
 
matlab
  

A(2,:) = [];  % 删除第2行
A(:,1) = [];  % 删除第1列
 
 
3. 矩阵自动扩展
 
直接给超出范围的索引赋值，自动补 0：
 
matlab
  

A(5,5) = 10; % 原3×3矩阵自动扩展为5×5，空位补0
 
 
 
 
四、矩阵拼接与分块
 
1. 水平拼接（左右合并）
 
 [A, B]  左右拼接，行数必须相同
 
matlab
  

A = [1 2; 3 4];
B = [5 6; 7 8];
C = [A, B]; 
 
 
2. 垂直拼接（上下合并）
 
 [A; B]  上下拼接，列数必须相同
 
matlab
  

D = [A; B];
 
 
3. 通用拼接函数
 
matlab
  

cat(1,A,B)  % 维度1：垂直拼接 等价 [A;B]
cat(2,A,B)  % 维度2：水平拼接 等价 [A,B]
 
 
 
 
五、矩阵形状变换
 
1. 重塑维度  reshape 
 
元素总数不变，强行修改行列数（列优先填充）
 
matlab
  

M = 1:12;
B = reshape(M, 3, 4); % 转为3行4列矩阵
 
 
2. 矩阵转置
 
matlab
  

A.'   % 非共轭转置（实数矩阵通用，推荐）
A'    % 共轭转置（复数会改变虚部，实数无区别）
 
 
3. 翻转与旋转
 
matlab
  

flipud(A)   % 上下翻转
fliplr(A)   % 左右翻转
rot90(A)    % 逆时针旋转90°
 
 
4. 降维挤压
 
matlab
  

squeeze(A)  % 删除单一维度，压缩高维矩阵
 
 
 
 
六、重中之重：矩阵运算 VS 逐元素运算
 
这是 MATLAB 最大易错点，必须严格区分。
 
1. 纯矩阵运算（线性代数规则）
 
遵循大学线性代数矩阵乘法规则：
 
符号 作用 维度要求 
 *  矩阵乘法 A列数 = B行数 
 /  矩阵右除 线性方程组求解 
 ^  矩阵幂 必须是方阵 
 
示例：
 
matlab
  

A = [1 2;3 4];
B = [5 6;7 8];
C = A * B;  % 标准矩阵乘法
 
 
2. 逐元素点运算（对应位置计算）
 
加 . ，两个矩阵行列尺寸完全一致，元素一一对应运算：
 
符号 作用 
 .*  元素相乘 
 ./  元素相除 
 .^  元素平方/幂次 
 .\  左除（少用） 
 
示例对比（核心区别）：
 
matlab
  

A.*B   % 每个位置元素单独相乘
A.^2   % 所有元素各自平方
 
 
补充： +   -  只有元素级运算，不需要加点。
 
 
 
七、线性代数专用矩阵操作
 
针对方阵/线性计算，工科高频使用：
 
matlab
  

det(A)     % 计算行列式
inv(A)     % 方阵求逆
pinv(A)    % 伪逆（非方阵/奇异矩阵）
rank(A)    % 矩阵的秩
trace(A)   % 矩阵的迹（对角线元素和）
eig(A)     % 求特征值、特征向量
norm(A)    % 矩阵范数
 
 
线性方程组求解
 
Ax=b
 
matlab
  

x = A \ b;  % 左除，MATLAB最优求解方式
 
 
 
 
八、矩阵统计与聚合运算
 
默认按列计算，可指定维度  dim ：
 
matlab
  

sum(A)        % 每一列求和
sum(A, 2)     % 按行求和

mean(A)       % 每一列平均值
max(A)        % 每一列最大值
min(A)        % 每一列最小值
cumsum(A)     % 累积求和
 
 
 
 
九、高维矩阵简单说明
 
三维/多维数组格式： A(行,列,页) 
 
matlab
  

T = rand(3,3,2); % 3行3列，2页三维矩阵
T(:,:,1)         % 取出第一页矩阵
 
 
 
 
十、高频易错总结
 
1. 索引从 1 开始，不要写  A(0,1) ；
2. 矩阵乘法用 * ，元素相乘必须用 .* ，否则维度报错；
3. 转置实数用 .' ，避免复数共轭错误；
4. 行列拼接时，必须保证行列维度匹配；
5. 冒号 : 、分号 ; 、逗号 ,  全部使用英文符号。
